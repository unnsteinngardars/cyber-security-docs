Acquisition Memory Triage NTFS

V3 reframes the course as an artefact catalogue. V2 walked through tools; V3 walks through evidence — every section opens with an Artefact Ledger that lists the concrete objects you can pull off a system, where they live, and which forensic questions they actually answer.

Each artefact is tagged with the kind of value it provides — timeline, user attribution, recovery, integrity, pivot, execution, credentials, persistence, or hidden-data exposure. The same chips repeat across sections so you can mentally connect findings across layers.

Open any tile below to dive into a section. Sections 01–10 are live; the remaining sections from V2 still apply unchanged for now.
